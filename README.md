# halfWay
gameJam模拟判案类游戏
开发时间：2025.8-2025.10
简介：玩家通过在不同地点收集证物，利用他们来指认嫌疑人，在法庭上与ai扮演的嫌疑人进行辩论，最后决定他们的下场，还受害者一个公道。
说明：在这个项目中，我实现了基本的主菜单选择关卡及设置相关的开发，主界面中地点切换，地点中证物的交互、展示、搜查官对话、搜集功能，证物的背包系统，后续证物的归类玩法，证物提示技能以及搜集成功与失败的整个闭环。其中，用到scriptObject来存储每个证物的相关信息，标签管理实现分类以及其他技术。此外还实现了搜证计时器，证物提示技能等玩法。
分工：此项目开始于2025.8的Unity*知乎gamaJam线下赛，团队共五人，三个美术同学，一个程序老师，我仅负责前期主菜单及证物搜集分类等说明中提到部分
